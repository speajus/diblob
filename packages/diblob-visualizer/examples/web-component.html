Usage

To use this web component in your application:

1. Install the package

npm install @speajus/diblob-visualizer

2. Import the web component

import '@speajus/diblob-visualizer/web-component';

3. Use it in your HTML

<diblob-visualizer 
  url="http://localhost:3001/events"
  mode="remote"
  updateInterval="1000">
</diblob-visualizer>

Attributes

  • url - The SSE endpoint URL
  • mode - "local" or "remote" (default: "remote")
  • updateInterval - Update interval in milliseconds (default: 1000)
